# gbtohtml5
A dodgy autoloading version of the amazing [JavaScript GameBoy Color Emulator](https://github.com/taisel/GameBoy-Online)

### How to use:
----------------
Open game.js and set the following

```js
game =
{
	"Sound": true,
	"RomBase64": "w2gA..."
}
```

Sound: toggles sound
RomBase64: base64 serialization of your game.gb rom
